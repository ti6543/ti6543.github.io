Gybing

Apparent Wind

Going Downwind

Before learning to gybe, we need to learn to go downwind.

This may seem simple, we can already steer. However, the problem is that as we go further off the wind, the apparent wind reduces.

The issue is that with zero or reversed Apparent Wind, we have nothing to balance against

For this reason, before gybing it is essential to be able to go downwind in control, balancing purely on the foil
